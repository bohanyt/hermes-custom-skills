### 🤖 Assistant

Tailscale itu semacam **VPN mesh network** yang bikin device-device kamu kayak satu jaringan lokal, meskipun beda-beda lokasi.

**Analogi simpel:**

Bayangin kamu punya laptop di rumah, HP di mana-mana, dan server di kantor. Normalnya mereka satu-satu, terpisah. Tailscale bikin mereka **semua nyambung ke jaringan virtual yang sama**, kayak colok ke switch yang sama — padahal fisiknya beda kota.

**Yang bikin beda dari VPN biasa:**

| | VPN biasa | Tailscale |
|---|---|---|
| Setup | ribet, perlu server | install → login → jadi |
| NAT traversal | sering gagal | pake **WireGuard + STUN**, hampir selalu tembus |
| IP | shared/public | **100.x.x.x** per device |
| Auth | password/cert | SSO (Google, GitHub, dll) |
| Mesh | hub-spoke | **full mesh** — device bisa langsung konek ke device |

**Contoh use case kamu:**

- Laptop kantor (172.20.10.10) jalanin Hermes dashboard
